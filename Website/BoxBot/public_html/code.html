Welkom in code-modus

Hier kan je de BoxBot besturen doormiddel van code
de code is simpel, je hebt forward, left en right, dit zijn de bewegingen.
Je moet ook aangeven hoelang en hoe snel de beweging moet zijn.
De tijd is in miliseconde(1000 miliseconde is één seconde).
De snelheid is maximaal 255 en minimaal 1; Je commando's zien er dus zo uit als het voorbeeld hieronder. 

richting;snelheid;tijd;

Na elk woord doe je een puntcomma, dat is het volgende teken ;. een voorbeeld is dus naar links naar voren en dan naar rechts, dat ziet er zo uit: 

forward;100;200;left;200;200;right;212;233;

De maximum snelheid is 255 en de maximum tijd is zo lang als je wilt.
Wanneer je de BoxBot wilt laten rijden druk je op de groene knop en wordt de knop rood.
Als je de BoxBot wilt stoppen druk je rode knop.
